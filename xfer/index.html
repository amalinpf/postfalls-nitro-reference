$getSPuserproperty Gets a user’s attribute (e.g., Manager, Email) from a Person/Group column. $getSPuserproperty(userField, propertyName) 
$getSPuserproperty([Requester|Requester], "Manager")
$getSPuserproperty([AssignedTo|Assigned To], "Email")
PF
Use on Person/Group fields; returns attributes like Email, Name, Manager, or Department.
Property names must match known AD/SharePoint attributes.

$strcat Concatenates values $strcat(value1, value2, ...) 
$strcat("Hello ", [Requester|Title])
WF
Joins static and dynamic text. No quotes needed around placeholders.

$substring Returns a portion of a string $substring(source, startIndex, length) 
$substring([Title|Title], 0, 10)
WF
Start index is zero-based; length is number of characters to return.
