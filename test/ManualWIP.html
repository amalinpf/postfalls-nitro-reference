String Functions

From: Crow Canyon NITRO Manual

Function Description Syntax Example
$Left Returns the leftmost characters of a string, given a length. $Left(string, length) $Left("PostFalls", 4) → "Post"
$Right Returns the rightmost characters of a string, given a length. $Right(string, length) $Right("PostFalls", 5) → "Falls"
$Substring Extracts a substring starting at a position, with a specified length. $Substring(string, start, length) $Substring("PostFalls", 5, 3) → "Fal"
$ToLowerCase Converts all characters in a string to lowercase. $ToLowerCase(string) $ToLowerCase("PostFalls") → "postfalls"
$ToUpperCase Converts all characters in a string to uppercase. $ToUpperCase(string) $ToUpperCase("PostFalls") → "POSTFALLS"
