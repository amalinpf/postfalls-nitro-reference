$getSPuserproperty Gets a user’s attribute (e.g., Manager, Email) from a Person/Group column. $getSPuserproperty(userField, propertyName) 
$getSPuserproperty([Requester|Requester], "Manager")
$getSPuserproperty([AssignedTo|Assigned To], "Email")
PF
Use on Person/Group fields; returns attributes like Email, Name, Manager, or Department.
Works in Workflows and Custom Actions when you need user details beyond the default field value.
Property names must match known AD/SharePoint attributes.

$getaduserproperty Returns an Active Directory attribute (e.g., Manager, Department) for a given user. $getaduserproperty(user, propertyName) 
$getaduserproperty([Requester|Requester], "Department")
PF
Accepts a Person/Group value or a username/email; returns the requested AD attribute.
Often combined with routing/approval logic.

🔤 String Functions
Function Description Syntax Example More Info
$strcat Concatenates values $strcat(value1, value2, ...) 
$strcat("Hello ", [Requester|Title])
WF
Joins static and dynamic text. No quotes needed around placeholders.
Comprehensive list – $strcat
$substring Returns a portion of a string $substring(source, startIndex, length) 
$substring([Title|Title], 0, 10)
WF
Start index is zero-based; length is number of characters to return.
Comprehensive list – $substring
$tolower Converts text to lowercase $tolower(source) 
$tolower([Requester|Title])
WF
Applies to string inputs; consider trimming first if needed.
Comprehensive list – $tolower
$toupper Converts text to uppercase $toupper(source) 
$toupper([Requester|Title])
WF
Useful for normalizing text for comparisons/sorting.
Comprehensive list – $toupper
$replace Replaces text in a string $replace(source, oldValue, newValue) 
$replace([Title|Title], "Issue", "Request")
WF
Replaces all occurrences of oldValue with newValue.
Comprehensive list – $replace
$regextract Extracts substring by regex $regextract(source, pattern, [modifier]) 
$regextract([Body|Body], @.*(Bob).*)
WF
Supports modifiers like HTMLDecode, HTMLStrip.
Comprehensive list – $regextract
$length Returns the length of a string $length(source) 
$length([Title|Title])
WF
Character count of the string value (not bytes).
